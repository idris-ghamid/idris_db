# idris DB Examples

This directory contains examples of how to use **idris DB**, the fastest NoSQL database for Flutter.

## 🚀 Quick Start
Check out [main.dart](main.dart) for a complete quickstart example.

- [Smart Logging](logging_example.dart): See how to track queries and transactions.
- [Arabic Support](arabic_support_example.dart): Implementation of bilingual error messages.
