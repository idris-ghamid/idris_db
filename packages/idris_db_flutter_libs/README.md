# IdrisDb Plus Flutter Libs

Flutter binaries for the [IdrisDb Plus Database](https://github.com/ahmtydn/IdrisDb). 

This package contains the native binaries required to run IdrisDb Plus on Flutter platforms (iOS, Android, macOS, Windows, Linux).

## Usage

This package is automatically included when you add `idris_db` to your dependencies. You don't need to use it directly.

```yaml
dependencies:
  idris_db: latest
  idris_db_flutter_libs: latest
```

For documentation and examples, please visit the [main repository](https://github.com/ahmtydn/IdrisDb).
