# البدء السريع

سيساعدك هذا الدليل على تثبيت وإعداد **إدريس دي بي (idris DB)** في مشروع Flutter الخاص بك.

## 1. إضافة الاعتمادات (Dependencies)

أضف ما يلي إلى ملف `pubspec.yaml`:

```yaml
dependencies:
  idris_db: ^1.0.3

dev_dependencies:
  idris_db_generator: ^1.0.3
  build_runner: ^2.4.0
```

## 2. تهيئة قاعدة البيانات

```dart
import 'package:idris_db/idris_db.dart';

void main() async {
  final idrisDb = await IdrisDb.open([UserSchema]);
  runApp(MyApp(db: idrisDb));
}
```

## 3. تنفيذ عملية كتابة

```dart
await idrisDb.writeTxn(() async {
  await idrisDb.users.put(newUser);
});
```
