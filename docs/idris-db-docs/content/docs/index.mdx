# Introduction

Welcome to the official documentation for **idris DB**, the high-performance NoSQL database for Flutter.

## What is idris DB?

idris DB is an enhanced version of Isar, optimized by IDRISIUM Corp for elite performance, advanced debugging, and first-class Arabic support.

## Key Features

- **Blazing Fast**: Up to 10x faster than traditional NoSQL solutions.
- **Type Safe**: No more runtime errors with schema-first code generation.
- **Query Analyzer**: Exclusive tool to optimize your database performance.
- **Smart Logging**: Production-grade logging system.
- **Bilingual**: Designed from the ground up for English and Arabic developers.
