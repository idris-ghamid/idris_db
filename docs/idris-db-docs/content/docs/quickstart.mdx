# Quickstart

This guide will help you install and set up **idris DB** in your Flutter project.

## 1. Add Dependencies

Add the following to your `pubspec.yaml`:

```yaml
dependencies:
  idris_db: ^1.0.3

dev_dependencies:
  idris_db_generator: ^1.0.3
  build_runner: ^2.4.0
```

## 2. Initialize Database

```dart
import 'package:idris_db/idris_db.dart';

void main() async {
  final idrisDb = await IdrisDb.open([UserSchema]);
  runApp(MyApp(db: idrisDb));
}
```

## 3. Perform a Write

```dart
await idrisDb.writeTxn(() async {
  await idrisDb.users.put(newUser);
});
```
